# 📦 Parcel Management Dashboard (Excel + VBA)

A secure, professional, and fully automated **Parcel Register & Reporting System** built using **Microsoft Excel with VBA**.

This project converts a standard Excel workbook into a **feature-rich parcel management application** suitable for offices, branches, mail rooms, and record-keeping departments where parcels must be **logged, tracked, searched, updated, printed, and protected** with proper controls.

---

## 🧭 Overview

The system replaces manual parcel registers with a **digital, searchable, secure, and printable solution** that behaves like real software while remaining lightweight and offline.

It provides:

* Structured parcel entry dashboard
* Protected parcel database
* Smart search with live results
* Update/Delete with password authorization
* Professional printable reports
* Emergency database repair utility
* VBA project protection

---

## ✨ Features

### 📝 Parcel Entry Dashboard

* Official, form-based data entry
* Mandatory field validation
* Calendar date picker (pure VBA)
* Automatic serial number generation
* One-click submit to database

Captured fields:

* Parcel Docket Number
* Date of Receipt
* Parcel Type
* Storage Location
* Remarks (optional)

---

### 🗂️ Protected Parcel Database

* Direct editing disabled
* Data written only via dashboard
* Auto-maintained serial numbers
* Structured and secured storage

---

### 🔎 Smart Search & Record Management

Search using:

* Docket number
* Parcel type
* Location
* Remarks
* Date range

From results:

* Select a record
* Update or Delete (password required)

---

### 🔐 Security Controls

* Hidden admin password (not readable in code)
* Masked password form
* VBA project locked for viewing
* Database sheet protected

---

### 🗓️ VBA Calendar Control

* Pure VBA calendar (no external dependency)
* Valid date selection only
* Month and year navigation

---

### 🖨️ Professional Print Dashboard

Generate reports using:

* Date range
* Search text
* Or both (strict filtering)

Print enhancements:

* Auto borders and alignment
* Wrapped text for long fields
* Dynamic print area up to last record
* Column headings repeat on each page
* Footer includes:

  * Date
  * Page number (Page X of Y)
  * Signature of In-Charge

---

### 🚑 Emergency Rebuild Utility

One-click macro to:

* Re-sort database by Date of Receipt
* Recreate Serial Numbers from 1 to last record
* Restore database order in emergencies

## 🔧 Setup Instructions

1. Open the Excel file and enable macros.
2. Use the **Parcel Entry Dashboard** to record parcels.
3. Use **Search Dashboard** to find, update, or delete records.
4. Use **Print_Dashboard** to generate reports.
5. Keep the VBA project locked to maintain integrity.
6. The Worksheets protection password is "123456".
7. Admin paswword to Update or Delete record is "admin123".
